西元年轉民國年
```sql
SELECT  GETDATE(),RIGHT( CONVERT(VARCHAR,DATEPART(YEAR,GETDATE())-1911),4)
```

複製：複製第一個參數，複製次數
```sql
SELECT REPLICATE('0',5) 
```

上面是用在例如補0，複製0，複製五次
