MacM1安裝pyodbc問題

先安裝unixodbc

brew install unixodbc

上面安裝的時候，注意版本路徑

export CPPFLAGS="-I/opt/homebrew/Cellar/unixodbc/2.3.9_1/include"

export LDFLAGS="-L/opt/homebrew/Cellar/unixodbc/2.3.9_1/lib"

pip安裝已經不能在M1上面使用

pip install pyodbc

5/13 安裝時出現的問題

目前ＯＤＢＣ不支援ＡＲＭ６４架構
